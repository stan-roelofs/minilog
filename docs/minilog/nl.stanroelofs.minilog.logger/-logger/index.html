A logger with a name/tag that writes logs using a Writer and Formatter.

Author

Stan Roelofs

Parameters

name

the name for this logger

level

the logging level, this will determine whether logs are actually written

writer

an object implementing the Writer interface which will be used to write log messages to an output

formatter

an object implementing the Formatter interface which will be used to format log messages
